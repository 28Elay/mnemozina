#  Mnemozina

Десктопный ежедневник с фоновыми уведомлениями в Telegram. Приложение позволяет создавать задачи, устанавливать точное время выполнения и получать напоминания прямо в мессенджер.

##  Возможности
- Создание задач с названием, описанием и точным временем.
- Переключение между тёмной и светлой темами интерфейса.
- Локальное хранение данных в базе SQLite.
- Фоновая проверка расписания и отправка уведомлений через Telegram-бота.
- Управление списком: отметка о выполнении и удаление задач.

##  Стек технологий
- **Язык**: Python 3.x
- **Интерфейс**: Tkinter
- **База данных**: SQLite3
- **Сеть**: `requests` (для взаимодействия с Telegram Bot API)
